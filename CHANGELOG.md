# Changelog

All notable changes to this project are documented in this file.

## [0.1.1] - 2026-06-24

### Changed

- Prepared the repository and release metadata for submission to the default HACS store.
- Corrected the refrigerator card preview alternative text.

## [0.1.0] - 2026-06-24

### Added

- Initial standalone LG ThinQ refrigerator card
- Automatic entity discovery by Home Assistant device ID
- Refrigerator and freezer target-temperature controls
- Door, Eco/Vacation, Express Cool and Quick Freeze status and controls
- Conditional LG ThinQ notification banner
- HACS packaging, Jenkins pipeline and GitHub Actions validation
